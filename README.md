# 📊 Excel & Power BI Data Analysis Projects

## 🚀 Project Overview

This project focuses on analyzing business data using Excel and Power BI to uncover meaningful insights and support data-driven decision-making.

The dashboards are designed to transform raw data into interactive visual reports that help identify trends, patterns, and performance metrics.

---

## ❓ Problem Statement

Businesses often struggle to:

* Understand large volumes of data
* Track performance metrics effectively
* Make quick, data-driven decisions

This project aims to solve these challenges by creating interactive dashboards that simplify complex data into clear visual insights.

---

## 🛠️ Tools & Technologies

* Microsoft Excel → Data Cleaning & Preparation
* Power BI → Data Visualization & Dashboard Creation

---

## 📁 Project Files

* `Excel_mini_projects.xlsx` → Dataset used for analysis
* `Mini_projects_PowerBi.pbix` → Power BI dashboard file

---

## 📊 Dashboard Features

* Interactive filters (slicers) for dynamic analysis
* Visualizations like bar charts, pie charts, and line graphs
* KPI indicators for performance tracking
* Clean and user-friendly dashboard layout
